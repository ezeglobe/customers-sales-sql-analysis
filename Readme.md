# Customer Sales Analysis Using MySQL

## Project Overview

This project explores a retail sales dataset using SQL in MySQL Workbench. The objective was to analyze sales performance, customer behavior, product performance, and regional trends by answering business questions through SQL queries.

## Database

Database Name: Customers Sales

## Tools Used

* MySQL Workbench
* SQL
* GitHub

## Business Questions Answered

### Overall Performance

* What is the total sales revenue?
* What is the total profit generated?
* How many orders were placed?

### Customer Analysis

* Who are the top customers by sales?
* How much revenue does each customer generate?
* How can customers be ranked based on sales performance?
* Which customers are profitable and which are loss-making?

### Segment Analysis

* Which customer segment generates the most sales?

### Product Analysis

* Which product categories generate the highest sales?
* Which sub-categories perform best?
* Which sub-categories perform worst?

### Regional Analysis

* Which regions generate the most sales?
* Which states generate the highest sales?
* Which cities are the top-performing markets?
