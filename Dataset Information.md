## Dataset Information

The dataset contains retail sales transaction data.

### Columns Used

* Order ID
* Customer ID
* Customer Name
* Segment
* Category
* Sub-Category
* Sales
* Profit
* Region
* State
* City

### Key Metrics Analyzed

* Total Sales
* Total Profit
* Number of Orders
* Customer Sales Performance
* Segment Sales Performance
* Category and Sub-Category Sales
* Regional Sales Performance
* State Sales Performance
* City Sales Performance
